# 概要
HTTPメソッドのGET/POST/PATCH/DELETEのリクエストを扱えるControllerを実装しました。
リクエストの内容は以下のとおりです。
- GET/POST: 名前、ID、生年月日を取得、登録
- PATCH/DELETE: 名前の更新、削除
